# Benchmark results

Generated by `python benchmarks/run.py`. All numbers are measured with the
public tiktoken BPE vocabularies on deterministic, seeded datasets defined in
`benchmarks/datasets.py`. Every HAAL rendering is round-trip-verified against
the source data before being counted. Baseline for savings: **compact JSON**
(no whitespace), the strongest common baseline.
